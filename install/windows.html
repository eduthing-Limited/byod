How to Install

Follow the steps below carefully. If you require assistance, please contact eduthing.

Windows

  1. Download the certificate file above.
  2. Locate the downloaded certificate file (usually in the Downloads folder).
  3. Open the Start menu and search for "Manage Computer Certificates".
  4. In the Certificates window, expand the "Trusted Root Certification Authorities" folder.
  5. Right-click on the "Certificates" folder and select "All Tasks" > "Import".
  6. Browse to the location of the downloaded certificate file and select it.
  7. Follow the prompts to import the downloaded certificate file.
  8. Once the import is complete, restart your web browser to apply the changes.

Check that everything worked

After installation, browse to a familiar HTTPS site. If you see a padlock and no warnings, SSL decryption is ready. Clear cache or restart your device if prompts persist.
