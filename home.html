Quick checklist

Follow these steps so your browsing remains protected and warning-free.

1) Download

Save the correct certificate to your device. If unsure, choose the .crt file first.

2) Install

Import the certificate into your device’s trusted authorities or profile store.

3) Trust

When prompted, allow the certificate for TLS/SSL so HTTPS filtering works.

4) Verify

Restart your browser and visit a site. If no warnings appear, you’re done.

Device-specific instructions

Choose your platform and follow the steps carefully. Screens may vary slightly by version.

Windows
Use the .crt file
  1. Double-click ns.crt and select Install Certificate.
  2. Pick Local Machine and choose Place all certificates in the following store.
  3. Select Trusted Root Certification Authorities → Finish → confirm prompts.
macOS
Use the .pem file
  1. Open netsweeper.pem and add it to login Keychain.
  2. Double-click the certificate in Keychain Access and set When using this certificate to Always Trust.
  3. Close the window and enter your password to apply.
iOS / iPadOS
Use the .pem file
  1. Download netsweeper.pem → tap Allow to add the profile.
  2. Go to Settings → Profile Downloaded → Install.
  3. Navigate to Settings → General → About → Certificate Trust Settings and enable full trust.
Android
Use the .crt file
  1. Download ns.crt and open it from the notification or Downloads.
  2. When prompted, name it (e.g., ISP SSL) and select VPN and apps or Wi-Fi for credential use.
  3. Confirm install. If requested, set your screen lock first.
ChromeOS
Use the .crt file
  1. Open chrome://settings/certificates → Authorities → Import.
  2. Select ns.crt and trust for Identify websites and Identify mail users.
  3. Save and restart Chrome.

Check that everything worked

After installation, browse to a familiar HTTPS site. If you see a padlock and no warnings, SSL decryption is ready. Clear cache or restart your device if prompts persist.
